## How to Use on Mobile

### Accessing Toolbar Controls

1. **Refresh dashboard** (F5 or pull down)
2. **Look at top toolbar** - it now has 2 rows:
   - **Top row:** Title + Edit/View toggle
   - **Bottom row:** All other controls (scrollable)
3. **Swipe left** on bottom row to see more buttons
4. **Tap any control** to use it

### Resizing Widgets

1. **Enable Edit Mode** (tap unlock icon)
2. **Select a widget** on canvas
3. **Look at bottom edge** of widget
4. **Press and drag** the bottom handle
5. **Widget height changes!**
6. **Release to set size**

### Switching Themes

1. **Scroll toolbar** to find sun/moon icon
2. **Tap icon** to toggle dark/light mode
3. **Instant theme change!**

### Creating New Dashboard

1. **Scroll toolbar** to find grid icon (right side)
2. **Tap "New Dashboard" button**
3. **Enter dashboard name**
4. **New dashboard created!**
