### Resizing Widgets
1. **Enable Edit Mode** (unlock button in toolbar)
2. **Look at bottom of any widget**
3. **See blue "Drag to resize" bar**
4. **Press and drag** up (shorter) or down (taller)
5. **Release** to set size

### Configuring Widgets
1. **Tap blue Settings button** (top-right of widget)
2. **Modal opens** with options:

   - Unit

### Duplicating Widgets
1. **Tap green Duplicate button**
2. **Instant copy** appears below original
3. **Drag to reposition** if needed

### Deleting Widgets
1. **Tap red Delete button**
2. **Widget removed immediately**
3. **Use Undo** (toolbar) if mistake

## Tips for Mobile

### Widget Sizing
- **Start small** (height 3) and resize up if needed
- **Swipe up** on resize handle to make shorter
- **Swipe down** to make taller
- **Full width** automatically on mobile

### Button Placement
- **Controls top-right** - won't block content
- **Resize bottom** - easy thumb access
- **Drag top-left** - standard location

### Best Practices
1. **One action at a time** - resize OR move, not both
2. **Tap precisely** on buttons (they're larger now)
3. **Use Settings** to configure before resizing
4. **Duplicate** instead of dragging new widgets (faster)
