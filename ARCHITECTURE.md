## Quick Reference Commands

### Start Everything
```bash
start_all.bat                    # Windows all-in-one
```

### Individual Components
```bash
# MQTT Broker
net start mosquitto              # Windows service

# Backend
cd backend
uvicorn main:app --reload

# Simulator
cd simulator
python esp32_mqtt_simulator.py

# Frontend
cd frontend
npm start
```

### Testing
```bash
# Test MQTT publish
mosquitto_pub -h localhost -p 1883 \
  -t "app/device/TEST/telemetry" \
  -m '{"telemetry": {"temp": 25}, "timestamp": "2025-12-19T12:00:00"}'

# Test MQTT subscribe
mosquitto_sub -h localhost -p 1883 -t "app/device/+/telemetry"
```
